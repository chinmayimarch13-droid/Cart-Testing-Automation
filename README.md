# Cart Testing Automation

## Project Overview

This project focuses on testing cart functionality in an e-commerce application using Manual Testing and Selenium Automation.

## Features Tested

* User Login
* Add Product to Cart
* Remove Product from Cart
* Verify Cart Items
* Verify Cart Count

## Manual Testing Activities

* Test Case Writing
* Functional Testing
* Positive and Negative Scenarios
* Bug Reporting

## Automation Testing Activities

* Selenium WebDriver Setup
* Element Identification using XPath
* Automated Cart Operations
* Validation of Cart Functionality

## Tools Used

* Python
* Selenium WebDriver
* GitHub

## Test Scenarios

1. Add a product to cart
2. Remove a product from cart
3. Verify cart count updates correctly
4. Verify selected product appears in cart

## Learning Outcomes

* Improved Selenium skills
* Learned cart functionality testing
* Practiced XPath and locators
* Gained hands-on automation experience
